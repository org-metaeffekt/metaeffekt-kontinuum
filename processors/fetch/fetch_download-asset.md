# Fetch - Download Asset

This mavenProcessor downloads a remote asset from a specified URL into a target directory.

## Properties

The different properties are sorted into three different groups which are explained in the top level [README](../../README.md)
of this repository.

### Input / Output
| Parameter        | Required | Description                                       |
|------------------|----------|---------------------------------------------------|
| output.asset.dir | yes      | The target directory to download the asset into.  |

### Parameters
| Parameter        | Required | Description                                       |
|------------------|----------|---------------------------------------------------|
| param.asset.url  | yes      | The URL of the remote asset to download.          |

### Environment
None.
