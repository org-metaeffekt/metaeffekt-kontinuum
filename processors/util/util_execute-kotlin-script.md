# Util - Execute Kotlin Script

This process executes a specified kotlin script file with a unspecified number of arguments.

## Properties

The different properties are sorted into three different groups.

### Input / Output
| Parameter                | Required | Description                                     |
|--------------------------|----------|-------------------------------------------------|
| input.kotlin.script.file | yes      | The kotlin script file to be executed.          |

### Parameters
| Parameter         | Required | Description                                                    |
|-------------------|----------|----------------------------------------------------------------|
| param.script.args | no       | Any additional arguments in the form of a space-separated list |

### Environment

None
